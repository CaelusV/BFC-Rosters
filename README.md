# BFCRosters
A complete compendium of all the registered Big Funny Cup teams and the associated images.

# How it works
For each player on a roster, different tags can be applied.
Tags: 
* (a) = active player
* (c) = captain
* (s) = silver medal player
* (g) = gold medal player

A separate [p=] command can be added to specify a unique name for the player portrait.
This is useful to optimize space for players with identical portraits.
Example: ```XXX01 +++ EXAMPLE NAME [p=sharedname] +++ GK```
